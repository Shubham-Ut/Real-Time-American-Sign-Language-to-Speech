# Real-Time-American-Sign-Language-to-Speech
Real-Time American Sign Language to Speech Conversion System Using Deep Learning

The Real-Time American Sign Language (ASL) to Speech Conversion System is a computer vision and deep learning-based application that recognizes static ASL alphabet gestures from live webcam input and converts them into spoken sentences. This project uses a Convolutional Neural Network (CNN) to accurately classify hand-sign images corresponding to the 26 letters of the English alphabet. Recognized letters are grouped into meaningful sequences which are then converted into audible speech using text-to-speech synthesis.

This system enables effective communication for deaf and hard-of-hearing individuals, acting as a natural interface between sign language gestures and audible language. It is designed for real-time performance, providing instant feedback through both visual display and synthesized voice output.

The core components of this project include:

Image Preprocessing & Augmentation: Efficient normalization and augmentation techniques to prepare the dataset for robust training.

Deep Learning-Based Classification: A Convolutional Neural Network trained to accurately classify ASL hand gestures.

Real-Time Recognition: Utilization of webcam input to capture hand gesture images on the fly.

Sentence Formation: Accumulation of letters captured during live inference to form words and sentences.

Text-to-Speech Conversion: Conversion of recognized text sequences into clear spoken output using the Google Text-to-Speech engine.

This system demonstrates the fusion of computer vision, machine learning, and natural language processing to solve real-world accessibility challenges.

👉 Dataset Used:
American Sign Language Alphabet Dataset (ASL Alphabet) — a publicly available dataset consisting of images for all 26 alphabet classes.
🔗: https://www.kaggle.com/datasets/debashishsau/aslamerican-sign-language-aplhabet-dataset
